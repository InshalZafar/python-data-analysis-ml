## Overview

This repository showcases **end-to-end Data Analysis and Machine Learning workflows** built using Python and industry-standard data science libraries. The projects demonstrate practical implementation of:

✔ Data Cleaning & Preprocessing  
✔ Exploratory Data Analysis (EDA)  
✔ Feature Engineering  
✔ Data Visualization  
✔ Classification Models  
✔ Model Evaluation & Performance Analysis  

The goal of this repository is to demonstrate **real-world machine learning pipelines**, transforming raw datasets into meaningful insights and predictive systems.

---

## Projects Included

### 1. Personality Dataset Classification
A complete machine learning pipeline for personality prediction using behavioral data.

#### Key Features
- Missing value handling
- One-Hot Encoding & Label Encoding
- Exploratory Data Analysis
- Feature importance analysis
- Classification using **Random Forest**
- Performance evaluation using:
  - Confusion Matrix
  - Precision
  - Recall
  - F1 Score
- t-SNE dimensionality reduction visualization

#### Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn

#### ML Workflow
```text
Data Collection
      ↓
Data Cleaning
      ↓
Feature Encoding
      ↓
Visualization & EDA
      ↓
Train-Test Split
      ↓
Random Forest Training
      ↓
Model Evaluation
```

### 2. Loan Approval Prediction
A machine learning classification system designed to predict loan approval outcomes based on applicant information.

#### Key Features
- Missing value imputation
- Data preprocessing
- Correlation heatmaps
- Feature standardization
- Multiple model comparison:
  - Random Forest
  - K-Nearest Neighbors (KNN)
  - Support Vector Classifier (SVC)
  - Logistic Regression

#### Best Model Performance
| Model | Accuracy |
|--------|----------|
| SVC | 80% |
| Logistic Regression | 79.6% |
| KNN | 74.6% |
| Random Forest | Overfit |

#### Technologies Used
- Python
- Scikit-Learn
- Pandas
- NumPy
- Matplotlib
- Seaborn
